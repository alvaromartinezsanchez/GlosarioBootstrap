Menu Horizontal Basico

Para crear un menú horizontal simple, añadimos la clase .nav a un elemento <ul>, seguido de la clase .nav-item para cada <li> y añadiendo la clase .nav-link a sus enlaces

Cada elemento del menú será un .nav-item, los cuales contienen un enlace tipo .nav-link a la sección a mostrar. Disponemos de la clase .disabled para deshabilitar elementos del menú, y de la clase .active para marcar el elemento del menú que está activo o seleccionado.

Barra de Navegación

Creación de la barra principal de navegación de nuestra web mediante la clase navbar.. Esta barra se adaptará al tamaño de pantalla, mostrando los elementos colapsados en un botón en pantallas pequeñas y de forma normal para pantallas más grandes. En las pantallas pequeñas, los elementos de navegación se colapsarían en un botón (togglerbutton). En caso de no usar la etiqueta <nav>, podemos crear la barra de navegación usando un<div> y añadiéndole el atributo role=“navigation” por cuestiones de accesibilidad.

Para crear esta barra en nuestro sitio web utilizaremos la etiqueta <nav>, que es la etiqueta de HTML5 que identifica un elemento de navegación

A esta etiqueta añadiremos las clases .navbar y.navbar-expand-* para indicar el tamaño a partir del cual la barra se mostrará de forma expandida. Por ejemplo, la etiqueta .navbar-expand-lg indica que la barra se mostrará en su tamaño completo a partir del tamaño depantalla grande (lg), apilándose verticalmente para tamaños más pequeños. Si queremos, lo podemos modificar cambiando el tamaño “lg” por otro de los posibles tamaños definidospor Bootstrap: “sm”, “md”, “lg” o “xl”.

<nav class="navbar navbar-expand-sm bg-light">

Barra de Navegacion Plegable

Podemos crear una barra de navegación plegable, muy utilizada especialmente en pantallaspequeñas, para ocultar los enlaces de navegación y reemplazarlos por un botón (togglerbutton) que los muestra al hacer clic sobre él

Para crear una barra de navegación plegable, haremos uso de un botón conclass="navbar-toggler", data-toggle="collapse" y data-target="#nombre" . La lista de enlaces tenemos que meterla dentro de una caja <div> con identificador“nombre”y las clases.collapsey .navbar-collapse, que definirá la zona que se colapsará (u ocultará) para pantallaspequeñas.

Estado activo / deshabilitado: puedes añadir la clase .active a un elemento para resaltar el enlace actual, o la clase .disabled para indicar que no se puede hacer clic en el enlace.

Anclaje de la barra de Navegacion

Bootstrap nos permite anclar o fijar la posición de la barra de tres formas distintas: fijarla a la parte superior añadiendo la clase .fixed-top a la etiqueta

, fijarla a la parte inferior con fixed-bottom. o usar el modo sticky (o pegajoso) con la clase .sticky-top, que anclarála barra a la parte superior mientras se realizascroll y cuando se alcanza el tope permanecerá fija

En los modos ".fixed-top" y ".fixed-bottom", dado que la barra se colocará de forma "flotante" sobre el contenido, es posible que oculte una parte del mismo. Para solucionar esto es necesario añadir un pequeño espaciado superior o inferior a la etiqueta . El alto de la barra es de 50px, por lo que se suele recomendar un espaciado de 70px, de la forma: body { padding-top: 70px; } /* En el caso de .fixed-top */ body { padding-bottom: 70px; } /* En el caso de .fixed-bottom */
