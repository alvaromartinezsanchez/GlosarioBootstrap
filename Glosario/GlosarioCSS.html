Cajas Flotantes

float

permite colocar los div uno junto al otro si hay espacio (tener en cuenta padding y margin para el tamaño total)

Cuando una caja se convierte en una caja flotante, se desplaza hasta la zona más a la izquierda o más a la derecha de la posición en la que originalmente se encontraba. Si se indica un valor “left”, la caja se desplaza hasta el puto más a la izquierda posible en esa misma línea (si no existe sitio en esa línea, la caja baja una línea y se muestra lo más a la izquierda posible en esa nueva línea). El resto de elementos adyacentes se adaptan y fluyen alrededor de la caja flotante. El valor “right” tiene una funcionamiento idéntico, salvo que en este caso, la caja se desplaza hacia la derecha. El valor “none” permite anular el posicionamiento flotante de forma que el elemento se muestre en su posición original.

Eliminar estilo

Display: block(modo bolque) o inline(en una linea) ( esta propiedad define el tipo de caja que genera un elemento. Todos los elementos generados en el HTML ocupan un espacio)

    Display:
  • none => se indica al elemento que no genere ninguna caja, con lo que el elemento simplemente desaparece, el elemento deja de ocupar un espacio en la presentación del documento, por lo que los elementos adyacentes simplemente ignoran su presencia y ocupan su espacio.
  • block =>elemento no permitirá la ubicación de elementos adyacentes, ocupará todo el ancho disponible dentro del elemento contenedor, y tendrá una altura que será determinada por el contenido,Cuando se determina el ancho, el elemento ocupará solo la extensión determinada, pero sin permitir la ubicación de elementos adyacentes, aún cuando haya espacio suficiente para que se ubique en él otro elemento.
  • inline => permiten la ubicación de elementos adyacentes ,No puede determinarse el ancho ni la altura de estos elementos, ya que como elementos en línea, las dimensiones de los mismos son determinadas por el contenido

Clase Hover

Las pseudo-clases :hover, :active, :link y :visited permiten al diseñador modificar el estilo de un elemento cuando el usuario realice una acción sobre el mismo. A continuación, se indican las acciones del usuario que activan cada pseudo-clase:

El orden de uso de estas pseudo clases es como se muestran a continuacion.!!

:link. se aplica a todos los vínculos que todavía no han sido visitados por el usuario.

:visited, se aplica a todos los vínculos que han sido visitados al menos una vez por el usuario.

:hover, el estilo creado se activa cuando el usuario sitúa el ratón encima de un elemento.

:active, el estilo creado se activa cuando el usuario pulsa con el ratón sobre un elemento. Se aplica durante un espacio de tiempo prácticamente imperceptible, ya que sólo dura desde que el usuario pulsa el botón del ratón hasta que lo suelta.

<p:hover { margin: 5px } >

Proceso de formado para el menu hover

Si queremos que todo el nav tenga un estilo lo hacemos en la etiqueta nav indicando color de fondo ancho y lo flotamos a la izquierda

Quitamos el estilo a nav ul(afecta al listado entero) => list-style: none;(quita efectos de lista) text-align:center; padding:0;

Seguimos por los elementos nav ul li(cada elemento li de la lista pero no al enlace de dentro) => display:block;(modo bloque) float:left;(flotado a la izquierda) Color de fondo negro y el resto de paddin, margin y border

Para cada uno de los enlaces nav ul li a(enlaces dentro de li dentro de ul dentro de nav) => text-decoration:none;(quitamos subrayado y otras) text-transform:uppercase;(Mayusculas) color:#white;(color letra)

Por ultimo añadimos el hover nav ul li a:hover => Cambiamos el fondo y el color de letra
