Botones

Estilos para botones (Colores)

Mediante la clase .btn podemos aplicar estilo a los elementos tipo button. Esta clase lapodemos combinar con .btn-primary, .btn-secondary, .btn-success, .btn-danger, .btn- warning, .btn-info, .btn-link, .btn-dark o .btn-light para crear botones para diferentes estados o acciones en nuestros formularios.

<button class=" btn " type="button">Basic</button>
<button class=" btn btn-primary " type="submit">primary</button>
<button class=" btn btn-secondary " type="reset">secondardary</button>

Estas clases no son exclusivas para las etiquetas "button" sino que también funcionarán de la misma forma con "a" y también con "input".

Link 

    <a class="btn btn-primary" href="#" role="button">Link</a>
    <input class="btn btn-primary" type="button" value="Input">

Botones Outline estilo distinto,sin fondo pero con un borde, con los botones outline.

<button class="btn btn-outline-primary" type="button">Boton Outline</button>


Tamaño

Podemos variar el tamaño de los botones simplemente añadiendo las clases .btn-lg, .btn-sm o .btn-block, 



    <button class=" btn btn-primary btn-sm" type="submit">Pequeño(sm)</button>
    <button class=" btn btn-primary" type="submit">Normal</button>
    <button class=" btn btn-primary btn-lg" type="submit">Grande(lg)</button>
    <br>
    <br>
    <button class=" btn btn-primary btn-block" type="submit">Block</button>

Disable

Podemos configurar el estado de un botón o un link a activo (con apariencia de estar presionado) conla clase.active , y configurarlo a un estado deshabilitado (sobre el que no se puede hacer click) mediante la propiedad disabled.

Link Active Link Disable 

    <button class="btn btn-primary active">Boton Active</button>
    <button class="btn btn-primary ">Boton Normal</button>
    <button class="btn btn-primary" disabled>Boton Disable</button>
    <a href="#" class="btn btn-primary active">Link Active</a>
    <a href="#" class="btn btn-primary disabled">Link Disable</a>
