Formularios

Estructura basica

debemos crear un form , y dentro añadir una clase .form-group para cada uno de los elementos de nuestro formulario.

                    <form action="">
                        <div class="form-group">
                            
                        </div>
                    </form>

Para permitir que Bootstrap ajuste correctamente el espaciado, cada bloque o grupo de unformulario (normalmente formado por una etiqueta label y algún elemento de entrada dedatos como un input, textarea, etc.) tendrá que estar agrupado por una caja contenedoracon la clase .form-group . Además, a cada input se le tiene que aplicar la clase .form_control .

Formulario Vertical:

                    <form action="">
                        <div class="form-group">
                            <label for="user">Login</label>
                            <input type="text" class="form-control" id="user" placeholder="User">
                        </div>
                         <div class="form-group">
                            <label for="passwd">Password</label>
                            <input type="password" class="form-control" id="passwd" placeholder="Pass">
                        </div>
                        <button class="btn btn-primary" type="submit">Sign in</button>
                    </form>

Formulario En Linea

Con la clase.form-inline aplicado en la etiqueta form podemos crear formularios que se dispondrán en una sola línea. En caso de no caber mas elementos los añadira en la siguiente linea


El codigo es igual que el de arriba pero añadiendo class="form-inline" en la etiqueta form

Ademas añadimos margenes (mr-2 y ml-2) a los imput y button para que queden bien

<form action="" class="form-inline" ></form>

Formulario Horizontal ( Uso de filas para cada form-group y columnas de tamaño ajustable con datos)

Mediante el uso del sistema de rejilla de Bootstrap podemos crear formularios con disposición horizontal. Para esto tendremos que crear una fila (.row) por cada grupo, y situar la etiqueta input cada uno en una columna.

Sign in

La etiqueta .row se añade al div de cada grupo, manteniendo también la etiqueta .form-group. Además, podemos aplicar la clase de lascolumnas para las etiquetas label directamente sobre dicho elemento, sin necesidad de crear una caja contenedora.

                        <form action="">
                            <div class="form-group row">
                                <label class="col-2 col-form-label" for="user">Login</label>
                                <div class="col-10">
                                    <input type="text" class="form-control" id="user" placeholder="User">
                                </div>
                            </div>
                            <div class="form-group row">
                                <label class="col-2 col-form-label" for="passwd">Password</label>
                                <div class="col-10">
                                    <input type="password" class="form-control" id="passwd" placeholder="Pass">
                                </div>
                            </div>
                            <div class="form-group row">
                                <div class="col-2">Sign in</div>
                                <div class="col-10">
                                    <button class="btn btn-primary" type="submit">Sign in</button>
                                </div>
                            </div>
                        </form>

Cuadros de Ayuda

Son textos que suelen acompañar a los input para informar al usuario sobre lo que debe escribir en dicho input. Los podemos añadir utilizando la etiqueta small y las clases .form-text y .text-muted.

!!..Esto es un cuadro de ayuda.....¡¡¡¡¡¡¡¡¡¡¡

Codigo:

                        <div class="form-group">
                            <label for="user">Usuario</label>
                            <input type="text" class="form-control" id="user" placeholder="Nombre de Usuario">
                            <small class="form-text text-muted">!!..Esto es un cuadro de ayuda.....¡¡¡¡¡¡¡¡¡¡¡</small >
                        </div>

Validacion de Formularios

Bootstrap proporciona clases, como .was-validated, que ayudan a la validación de unformulario, dando al usuario información (feedback) sobre los campos que deben ser completados antes de ser enviado. Los campos de entrada tendrán un borde verde (válido)o rojo (no válido) para indicar lo que falta en el formulario. Ta m b i é n puede visualizarse un mensaje utilizando las clases .valid-feedback o .invalid-feedback para decirle al usuariolo que falta o debe hacerse antes de enviar el formulario.

Por favor, rellene este campo

Por favor, rellene este campo

                    <form action="" class="was-validated">
                        <div class="form-group">
                            <label for="user">Usuario</label>
                            <input type="text" class="form-control" id="user" placeholder="Nombre de Usuario" required>
                            <div class="valid-feedback">Valido</div>
                            <div class="invalid-feedback">Por favor, rellene este campo</div>
                        </div>
                        <div class="form-group">
                            <label for="passwd">Contraseña</label>
                            <input type="password" class="form-control" id="passwd" placeholder="Contraseña" required>
                            <div class="valid-feedback">Valido</div>
                            <div class="invalid-feedback">Por favor, rellene este campo</div>
                        </div>
                        <div class="form-group form-check">
                            <label class="form-check-label">
                                <input type="checkbox" class="form-check-input" required>Acepto politicas de privacidad
                                <div class="invalid-feedback">Marca esta casilla para continuar</div>
                            </label>
                        </div>
                        <button class="btn btn-primary" type="submit">Entrar</button>
                    </form>
