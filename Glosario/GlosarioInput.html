Inputs

Suelen utilizarse dentro de un formulario <form>

Texto

<div class="form-group">
    <label for="texto">Login</label>
    <input type="text" class="form-control" id="texto" placeholder="Escribe tu texto..">
</div>

Password

<div class="form-group">
    <label for="pwd">Password</label>
    <input type="password" class="form-control" id="pwd" placeholder="Pass">
</div>

Textarea

<div class="form-group">
    <label for="comment">Comentarios</label>
    <textarea class="form-control" rows="5" name="comment" id="comment">Escribe tus comentarios...</textarea>
</div>

Lista Select

<div class="form-group">
    <label for="lista">Lista Select</label>

    </select>
</div>

Tamaño de los Imput

Podemos cambiar el tamaño de los ítems añadiendo los sufijos –sm o –lg a la clase .form-control

                <form action="">
                    <div class="form-group">
                        <input type="text" class="form-control form-control-sm" name="text1" placeholder="Input de tamaño pequeño">
                    </div>
                    <div class="form-group">
                        <input type="text" class="form-control " name="text2" placeholder="Input de tamaño por defecto">
                    </div>
                    <div class="form-group">
                        <input type="text" class="form-control form-control-lg" name="text3" placeholder="Input de tamaño grande">
                    </div>
                </form>

Imput de solo lectura o desactivados

Mediante la propiedad readonly podemos asignar el estado de solo lectura a un item, de modo que el contenido del item no pueda ser modificado. Tambien podemos desactivar un item mediante el atributo disabled

                <form action="">
                    <div class="form-group">
                        <input type="text" class="form-control" name="text1" placeholder="No puedes modificar este texto(solo lectura)" readonly><!--Con readonly Los datos de Este campo si se envia con el formulario-->
                    </div>
                    <div class="form-group">
                        <input type="text" class="form-control" name="text2" placeholder="Campo Desactivado" disabled><!--Con disabled Los datos de Este campo NO se envia con el formulario-->
                    </div>
                    <div class="form-check">
                        <label class="form-check-label">
                            <input type="checkbox" class="form-check-input" checked disabled>Acepto condiciones y terminos de uso
                        </label>
                    </div>
                </form>

Agrupar inputs con otros elementos

Podemos añadir texto o botones al principio,final o a ambos lados de un campo tipo imput.Para esto tenemos que agrupar dicho input dentro de un imput-group y añadir dentro del grupo el elemento que queremos agrupar con la etiqueta input-group-prepend (delante del imput) o con la etiqueta imput-group-append (detras del imput) y tambien con la clase imput-group-text puedes añadir estilo al texto de ayuda

@
@ces-vegamedia.es
@gmail.com 

                <form action="">
                    <div class="input-group mb-3">
                        <div class="input-group-prepend">
                            <span class="input-group-text">@</span>
                        </div>
                        <input type="text" class="form-control" placeholder="Usuario">
                    </div>
                    <div class="input-group mb-3">
                        <input type="text" class="form-control" placeholder="Usuario">
