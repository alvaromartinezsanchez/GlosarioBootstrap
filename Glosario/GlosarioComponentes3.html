Alertas

En Bootstrap las alertas se crean con clase.alert seguida de una de las clases contextuales:

  • .alert-sucess(Verde)
  • .alert-info(Azul)
  • .alert-warning(Naranja)
  • .alert-danger(Rojo)
  • .alert-primary(Azul)
  • .alert-secondary(Gris)
  • .alert-light(Transparente)
  • .alert-dark.(Negro/Gris)
Success Este cuadro de alerta indica una accion positiva o exitosa 

<div class="alert alert-success">
	<strong>Success </strong>Este cuadro de alerta indica una accion positiva o exitosa
</div>


Enlaces dentro de Alertas

Podemos agregar enlaces dentro de un cuadro de alerta mediante la clase .alert-link, creándose enlaces de colores coincidentes.

Peligro Cuidado con el contenido. Pinche aqui para mas detalles 

<div class="alert alert-danger">
	<strong>Peligro </strong>Cuidado con el contenido.
	<a href="#" class="alert-link">Pinche aqui para mas detalles</a>
</div>


Cerrar Mensaje Alerta [X]

Paracerrar un mensaje de alerta, debemos añadir la clase .alert-dismissibleal contenedorde alertas y definir un elemento tipo botón o link al que agregamos las propiedades class=“close” y data-dismiss=“alert”. El cuadro de alerta desaparecerá cuando hagamosclick sobre este elemento (en HTML suele utilizarse el icono× (×) para los botones decierre, en lugar de la letra “x”).

Haz click sobre el simbolo "x" a la derecha para cerrarme

<div class="alert alert-success alert-dismissible">
	<button type="button" class="close" data-dismiss="alert">×</button>
	Haz click sobre el simbolo "x" a la derecha para cerrarme
</div>

Cards

Una tarjeta o card es una caja bordeada con algo de relleno alrededor de su contenido.Incluye opciones para encabezados, pies de página, contenido, colores, etc.

Para crear una tarjeta básica utilizaremos la clase .card, y para el contenido dentro de la tarjeta, la clase card-body:.

La clase card-header añade un header a la tarjeta, y la clase .card-footer añade un footer

Collapse

Los elementos de tipo plegable o collapse son útiles cuando queremos ocultar o mostraruna gran cantidad de contenido.Bootstrap define la clase .collapsepara crear elementosplegables y cuyo contenido se mostrará u ocultará con solo hacerclic en un botón.

Para crear mostrar/ocultar contenido contraíble (se encuentra dentro de un <div>) , podemos utilizar tanto un elemento <a> comoun elemento <button> , al que añadiremos el atributo data-toggle=“collapse” . Luego debemos conectar estos elementos con el contenido plegable (a través de un id que se asigna al <div> donde está el contenido), en el caso de botonescon el atributo data-target y para enlaces con el atributo href .

Lorem ipsum dolor sit amet, consectetur adipisicing elit.
Lorem ipsum dolor sit amet, consectetur adipisicing elit.

Por defecto, el contenido plegable está oculto. Agregando la clase .show podemos hacerque el contenido se muestre de manera predeterminada:

<button type="button" class="btn btn-primary" data-toggle="collapse" data-target="#plegable1" >Plegable boton</button>
<div id=" plegable1 " class=" collapse">
	Lorem ipsum dolor sit amet, consectetur adipisicing elit.
</div>
<button href="#plegable2" class="btn btn-primary" data-toggle="collapse">Plegable enlace</button>
<div id="plegable2" class="collapse show">
	Lorem ipsum dolor sit amet, consectetur adipisicing elit.
</div>

Carousel

Uncarousel es una presentación (slideshow) de imágenes que se van mostrando de formacíclica (después de la última volverá a mostrarse la primera)

Contenedor:

Un carousel se define con la clase “.carousel slide”

<div class="carousel slide" id="ejemplo-carousel" data-ride="carousel"></div>

  • Una primera parte (obligatoria) que contiene las imágenes que forman la presentación.

    • clases “.carousel-inner” para el contenedor y “.carousel-item” para cada una de las imágenes

    atributo “active”.

    Atención a los tamaños de las imágenes. Podemos utilizar los atributos height y width para indicar el alto y ancho con que queremos que se muestre.

    También podemos hacer uso de la clase “img-fluid” para hacer nuestra web responsive.

    <div class="carousel-inner">
	<div class="carousel-item active">
		<img src="./img/banner.jpg" alt="Imagen1" class="img-fluid" width="100%">
	</div>
	<div class="carousel-item ">
		<img src="./img/banner2.jpg" alt="Imagen2" class="img-fluid" width="100%">
	</div>
	<div class="carousel-item ">
		<img src="./img/banner3.jpg" alt="Imagen3" class="img-fluid" width="100%">
	</div>
</div>

  • Una segunda parte (obligatoria) con los controles que nos permite desplazarnos hacia la izquierda y derecha.

    • izquierda (clase “.carousel-control-prev”) y derecha (clase “.carousel-control-next”).

    <a class="carousel-control-prev" href="#ejemplo-carousel" data-slide="prev">
	<span class="carousel-control-prev-icon"></span>
</a>
<a class="carousel-control-next" href="#ejemplo-carousel" data-slide="next">
	<span class="carousel-control-next-icon"></span>
</a>

  • Una tercera parte (opcional) con los indicadores de las imágenes para saber por cuál delas imágenes vamos. 

    • <ul class="carousel-indicators">
	<li data-target="#ejemplo-carousel" data-slide-to="0" class="active"></li>
	<li data-target="#ejemplo-carousel" data-slide-to="1"></li>
	<li data-target="#ejemplo-carousel" data-slide-to="2"></li>
</ul>

Y podemos añadir un título a cada imagen haciendo uso de la clase .carousel-caption 

<div class="carousel-item active">
	<img src="./img/banner.jpg" alt="Imagen1" class="img-fluid" width="100%">
	<div class="carousel-caption">
		<h4>Titulo</h4>
		<p>Texto....</p>
	</div>
</div>

Ventana Modal

Un componentemodal es una ventana o cuadro de diálogo que permanece oculta cuandose visualiza la página en el navegador y se muestra al pulsar sobre un enlace o un botón

Lo más importante del código para la ventana Modal es que el DIV principal (con clase “.modal”) tenga un ID (en el ejemplo, “myModal”). El botón que muestra la ventana Modal es un botón normal al que debemos añadir los atributos “data-toggle” y “data-target”, este último es el más importante, porque le asignamos el valor del nombre que le hemos dado al ID del DIV principal de la ventana Modal.

Titulo del Modal

Contenido del Modal

<button type="button" class="btn btn-primary" data-toggle="modal" data-target="#myModal">Abrir Modal</button>
<!--Ventana Modal-->

	<div class="modal-dialog modal-xl">
		<div class="modal-content">

				<h4 class="modal-title">Titulo del Modal</h4>

			<!--Modal Body-->
			<div class="modal-body">
				<p>Contenido del Modal</p>
				<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit...</p>
			</div>
			<!--Modal footer-->
			<div class="modal-footer">
				<button type="button" class="btn btn-danger" data-dismiss="modal">Cerrar</button>

		</div>
	</div>
</div>
