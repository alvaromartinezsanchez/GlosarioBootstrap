Tamaños de columna

Clase Dimensiones
.col < 576 px
.col-sm ≥ 576 px
.col-md ≥ 768 px
.col-lg ≥ 992 px
.col-xl ≥ 1200 px

Gird

Alineacion Fila
Vertical

class= align-item-(start,center,end) Alinea las columnas arriba,centradas,abajo dentro de una fila

Horizontal

class= justify-content-(start,center,end,around,between) around= espacio a ambos lados between=espacio entre columnas.

Alineacion Columna
Vertical

class= align-self-(start,center,end) Alinea cada columna individualmente arriba,centradas,abajo

Horizontal

Espaciado

offset-(tamaño)-(espacios) Deja un espacio a la izquierda de la columna indicada class=” offset-sm-5” (max.11 espacios) offset-0 elimina el offset(si filtramos por tamaño)

Margenes

mt,mr,mb,ml Añade margen a la columna Class=”ml-10”(margen izquierdo de 10) ó “ml-sm-10”(para tamaño sm) ó “ml-sm-auto”(ajusta según el tamaño de las columnas en la fila y el espacio libre)

Anidamiento

Anidamiento: .container>.row>.col  .row>.col .col .col .row>.col .row Podemos anidar dentro de una row muchas col y dentro de una de esas col otra row con mas col (lo mejor es hacer muchas filas y dentro sus columnas correspondientes asi podemos gestionar por filas y luego por columnas).

Ancho de columna Variable

col-auto ó col-(tamaño)-auto Ajusta el tamaño de la columna al texto. class=”col-auto”

col Ajusta el tamaño a la cantidad de columnas existentes dentro de la fila(se pueden meter mas de 12 columnas),mientras no haya un salto de fila forzado.(w-100 ó )

Forzar Cambio de fila

w-100 class=”w-100”. Realiza un cambio de fila, se puede convinar con d-block y d-none para alterar la estructura según nos convenga por tamaños.

Mostrar Ocultar (Filas y Columnas)

d-none Oculta el elemento class=”d-none d-sm-block d-xl-none”

d-block Muestra elemento

Ordenar

.order-($) Podemos indicar el orden de cada columna dentro de una fila,filtrando por tamaños también. class=”col order-(tamaño)-(1,2,..12)”
