Videos

Para un solo video(se indica la direccion en src y los atributos necesarios

< video src"direccion/archivo.mp4" atributos > </video>

Atributos:

  • src = indica la ruta donde se encuentra el vide o a insertar
  • controls = incorpora controles tipicos (reproducir,pantalla completa...)
  • width = ancho del video
  • height= alto del video
  • loop = reproduce continuamente el video en modo bucle
  • poster="direccionImagen" => muestra una imagen previa a la reproduccion del video
  • autoplay = fuerza la reproduccion del video cuando le sea posible al navegador
  • preload = gestiona la cache, pudiendo tomar los siguientes valores:
    1. auto : sugiere la descarga del archivo cuanto antes (valor por defecto).
    2. none: indica que el vídeo no debería ser cacheado, por lo general con el propósito de minimizar tráfico innecesario.
    3. metadata: indica al navegador que trate de obtener información sobre la fuente (dimensiones, duración, primer cuadro…)

Varios videos:

Sustituimos la etiqueta src por source src(para cada video) entre las etiquetas de apertura y cierre de video

<video whidth="400" controls autoplay loop poster="direccion/archivo.jpg">
	<source src="direccion/archivoVideo.jpg">
	<source src="direccion/archivoVideo2.jpg">
</video>

En caso de que los videos esten almacenados en internet creamos un iframe src="direccionCompleta" whidth="anchura" height="altura" id="id que youtube o la pagina de videos aya asignado al video"

Recuerda sustituir la etiqueta watch de la direccion de youtube por embed

<iframe src="https://www.youtube.com/embed/nPIPHV26awI" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

(1=activo 0=no activo)Podemos añadir los atributos ?autoplay=1 ?loop=1 y ?controls=0 justo despues de indicar el id del video

src="https://www.youtube.com/embed/467a8x78954?autoplay=1?loop=0?controls=1"

Tambien podemos pulsar sobre el video con boton derecho, estadisticas para nerds y hacemos clic derecho dentro copiar codigo de insercion (recuerda eliminar altura y anchura por defecto)

FORMULARIOS

<form>

Propiedades:

  • method="" => Metodo de envio, por defecto POST
  • action="" => Pagina a la que me dirige o direccion de la pagina
  • name="" => Nombre del formulario

Etiquetas:

Input

<input type="text" name="texto1" maxlength="10" value="Nombre_Usuario">

  • name => nombre del elemento
  • type => Tipo de elemento
    • text => texto (propiedades: name=> nombre, maxlenght => nº max de caracteres, size=> tamaño del campo
    • password => contraseña (propiedades: name=> nombre, maxlenght => nº max de caracteres, size=> tamaño del campo
    • hidden => dato oculto (propiedades name(nombre) maxlength(max caracteres)
    • radio => radio buton (propiedades: name(nombre) value(valor))
    • checkbox => checkbox (propiedades: name(nombre) value(valor))
    • button =>boton (propiedades: name(nombre) value(valor))
    • submit =>botonEnvio (propiedades: name(nombre) value(valor))
    • reset => borra valores (propiedades: name(nombre) value(valor))

Select

		<form method="post" action="pagina.php">
			<select name="caballos" >
				<option>60 caballos</option>
				<option>10 caballos</option>
				<option>115 caballos</option>

Atributos:

  • option => contiene los valores del select
  • size => numero de elementos que muestra
  • multiple => permite seleccionar mas de un valor de la lista

Text Area

			<h3>Escribe un comentario de tu peli favorita</h3>

propiedades:

  • cols => nº de columnas del area de texto
  • rows => nº de filas
